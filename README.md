# mips-pipeline-processor-vhdl-fpga
Design and implementation of a pipelined MIPS processor in VHDL for FPGA, including the main pipeline stages and control logic.
The implemented processor executes a program that processes an array of numbers stored in memory and determines the maximum even value among them.
## Pipeline Stages
- Instruction Fetch (IF)  
- Instruction Decode (ID)  
- Execute (EX)  
- Memory Access (MEM)  
- Control Unit  
## Technologies

- VHDL  
- FPGA design  
- Vivado  
- MIPS architecture
